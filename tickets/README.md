# Ticket System

A dependency-free starter ticket system that runs in the browser.

## What works now

- Create, edit, and delete tickets
- Track status, priority, customer, assignee, tags, and notes
- Filter by search, status, and priority
- View queue metrics and status columns
- Save tickets in browser local storage
- Export and import tickets as JSON

## Run it

Open `index.html` in a browser.
